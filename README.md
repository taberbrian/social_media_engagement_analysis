# Unannounced App: Social Media Engagement Analysis

This repository contains the data and R Markdown script used to analyze cross-platform social media engagement trends. The insights from this project directly inform the UX, UI, and feature development for an **Unannounced App**.

## View the Live Project
[Click here to view the interactive HTML dashboard](https://taberbrian.github.io/social_media_engagement_analysis/)

## Data Source
The dataset used in this analysis (`social_media_viral_content_dataset.csv`) was sourced from Kaggle by Ali Hussain.
